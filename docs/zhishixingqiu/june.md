### 沉默王二-Java 程序员进阶之路六月第二周打卡总结

Java 程序员进阶之路的网站服务器被攻击后，一直在犹豫要不要重装系统。重装吧，怕影响网站的流量，不重装吧？服务器内存占用一直飙升到 80%，很耽误事。

那趁这个周末吧，流量少的时候还是重装一下吧，因为 80% 的内存占用率，基本上没有多少事情可以做。反正里面主要就是一个 Nginx 服务器和一个 Git 服务。就能把 Java 程序员进阶之路跑起来了。
